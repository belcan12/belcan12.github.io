Projekt (case-studies)

My Movie Database — Filmdatabas

HTML, CSS, JavaScript

Webapp som efterliknar IMDB: sök filmer, se detaljerade filmsidor och spara favoriter. Hämtar startinnehåll (trailers + topplista) från ett film-API och använder OMDb API för både breda sökningar och imdbID-detaljer. Fokus på tillgänglighet (ALT, semantik) och responsiv design.

Where it’s @ — Events API (Serverless)

Node, Serverless, AWS

Händelse-/biljett-API på AWS: endpoints för event (skapa, lista, uppdatera), biljettlogik och enkel sök/paginering. Data i DynamoDB, API Gateway + Lambda och statiska resurser i S3. Fokus på validering, robust felhantering och ren endpoint-struktur som är lätt att dokumentera och testa.

API GatewayLambdaDynamoDB

GitHub: https://github.com/belcan12/whereitsat2

Bonz.ai — Hotel Booking API

Node, Serverless, DynamoDB

Boknings-API för hotell: sök lediga rum efter datum, skapa/avboka bokningar och beräkna pris med regler (t.ex. säsong/helg). Node/Serverless med DynamoDB. Fokus på datum-edge cases, inputvalidering och tydliga JSON-svar som är enkla att konsumera i en frontend.

GitHub: https://github.com/belcan12/bonzai

FlappyBird2 — 2D-spel

JavaScript/Kotlin (grundläggande), spel-loop

Flappy Bird-klon från studierna: game-loop, gravitation/hoppfysik, hinder som genereras löpande, kollisionsdetektering och poäng med restart-flöde. Fokus på input-hantering, tidsstyrd uppdatering och tydlig separation mellan logik och rendering.
